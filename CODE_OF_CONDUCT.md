# Contributor Code of Conduct

This project follows the Contributor Covenant Code of Conduct. We expect all contributors and
community members to be respectful, inclusive, and constructive.

Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting
the project maintainers at: krystedongmouo@gmail.com. All reports will be handled with
confidentiality and care.

If you have questions about this policy or how it is enforced, please reach out to the same contact.

For the full text of the Code of Conduct, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
